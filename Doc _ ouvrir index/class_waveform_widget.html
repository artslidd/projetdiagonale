Detailed Description

Classe d'interface utilisateur s'occupant de gérer la waveform en haut à droite de l'écran. Cette classe ne dessine pas la waveform, c'est le travail de WaveformPixmap. En revanche, elle gère les interactions avec les widgets de la partie affichage de waveform du logiciel. Elle répercute les demandes de l'utilisateur sur la WaveformPixmap.
