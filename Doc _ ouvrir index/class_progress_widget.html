Detailed Description

La classe ProgressWidget se contente d'ajouter à une QProgressBar les fonctions de base nécessaires pour servir de barre de navigation. En particulier, il est possible de cliquer sur la barre pour changer l'avancement de la lecture d'un son. Il ne s'agit pas d'une simple ProgressBar cliquable, mais bien d'une barre de navigation gérant une longueur en secondes et travaillant sur des temps. Le widget peut optionnellement être associé à un label indiquant la progression actuelle.
