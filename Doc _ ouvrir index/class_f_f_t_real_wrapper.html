Detailed Description

Wrapper around the FFTRealFixLen template provided by the FFTReal library

This class instantiates a single instance of FFTRealFixLen, using FFTLengthPowerOfTwo as the template parameter. It then exposes FFTRealFixLen<FFTLengthPowerOfTwo>::do_fft via the calculateFFT function, thereby allowing an application to dynamically link against the FFTReal implementation.

See http://ldesoras.free.fr/prod.html
