Detailed Description

Ce widget est une QGroupBox disposant d'un menu à gauche permettant d'afficher plusieurs onglets. Il est utilisé pour sélectionner une source. On place alors à l'intérieur un stack widget pour simuler l'effet d'onglets. En bas à gauche du widget se trouve un petit bouton MIDI relié à un signal.
