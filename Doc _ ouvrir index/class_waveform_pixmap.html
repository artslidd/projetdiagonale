Detailed Description

Classe permettant d'afficher une waveform à l'écran. Cette classe reçoit des informations sur l'affichage (synchro, période) ainsi que des trames successives. A partir de là, elle crée en interne une pixmap qui peut être dessinée à l'aide de la fonction draw().
