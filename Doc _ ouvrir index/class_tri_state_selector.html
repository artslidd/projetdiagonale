Detailed Description

Le TriStateSelector est un widget très simple affichant trois petits ronds cliquable. Chaque rond correspond à un état et est actualisé soit via le slot setState(), soit après un reset(), soit suite à un clic. Si l'utilisateur clique sur un rond, l'état change et un signal est émis.
