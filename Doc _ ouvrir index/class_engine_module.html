Detailed Description

Classe abstraite représentant une source audio dans le programme. Chaque classe fille doit pouvoir demander à s'activer (quand on appuie sur un élément de son interface graphique), et doit pouvoir gérer son activation effective tout comme sa désactivation. Un autre prérequis pour être un module est de pouvoir superposer à sa sortie audio des sinus lors de l'appui sur des touches de piano.
