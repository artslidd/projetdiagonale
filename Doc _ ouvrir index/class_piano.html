Detailed Description

La classe affichant le piano et gérant les clics sur ses touches. Cette classe peut selon son mode afficher soit un piano, soit une image générique (tortue/lièvre). En mode piano, si l'utilisateur clique sur une touche, la classe envoie des signaux en conséquence.
