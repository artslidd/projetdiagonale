Detailed Description

La classe FilterWidget permet à l'utilisateur de configurer le filtre quand il le souhaite. Visuellement, cette classe est le rectangle dans lequel est dessiné grossièrement le gabarit du filtre. En interne, c'est elle qui centralise tous les clics sur les widgets associés au filtrage (boutons radio, slider, dial...) et qui configure le filtre m_filter() en conséquence.
