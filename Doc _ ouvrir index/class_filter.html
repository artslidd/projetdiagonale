Detailed Description

Classe s'occupant de filtre le signal audio dans la fonction inOut de l'EngineModule actif Cette classe représente un filtre donné au sens traitement du signal. Elle fournit trois types de fonctions. Le premier groupe de fonctions permet de configurer le filtre (fréquences caractéristiques, amplitude bande coupée, atténuation globale de -6dB). Le second groupe permet d'afficher le filtre en fournissant de quoi savoir comment le tracer. Le dernier groupe permet d'effectivement appliquer le filtre sur une trame.
