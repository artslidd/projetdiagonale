Detailed Description

Classe d'interface utilisateur gérant la panneau de configuration MIDI. Cette classe ne fait que traduire les actions de l'utilisateur en commande pour le moteur MIDI MidiEngine. Elle n'effectue aucun traitement elle-même. Son objectif est notamment de permettre à l'utilisateur d'associer à chaque bouton de son périphérique MIDI un rôle dans l'application.
